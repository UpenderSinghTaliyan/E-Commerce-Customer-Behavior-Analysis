# E-Commerce-Customer-Behavior-Analysis
This project analyzes customer behavior on an e-commerce platform to uncover key factors influencing purchase decisions and membership patterns. The dataset includes 1000 customer sessions with features like age, device type, session duration, pages viewed, purchase amount, and membership level.
Through Exploratory Data Analysis (EDA), several insights were identified. Tablet users showed the highest purchase amount, while customers aged 36–45 were the most active. Interestingly, pages viewed had a weak relationship with spending, indicating that engagement quality matters more than quantity. Desktop users had the lowest session duration, while non-members spent more time browsing. Silver members had the highest average purchase value, whereas Platinum was the most popular membership tier.
Statistical analysis further validated findings. The T-test showed no significant difference in spending between younger and older customers, and the Chi-square test indicated that device type and membership level are not strongly related.
Machine learning models were applied to classify customers. Logistic Regression improved from 25% to 55% accuracy after feature engineering (engagement score, pages per minute), while KNN achieved 72% accuracy using optimized K value.
Overall, the project highlights that customer behavior is influenced more by engagement patterns and membership strategies than basic demographics. These insights can help businesses improve targeting, user experience, and data-driven decision-making.
